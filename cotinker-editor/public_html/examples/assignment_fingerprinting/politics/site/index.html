Velkommen!

Velkommen til vores hjemmeside om samfundsfag! Her vil du finde information om de fire undersider: sociologi, politik, økonomi og international politik.

Sociologi er en videnskab, der undersøger samfundets strukturer og processer, herunder menneskelige relationer, kulturelle normer, magtforhold og sociale grupper.

Politik handler om, hvordan samfundet er organiseret og styret. Det omfatter både indenrigspolitik og udenrigspolitik, og det kan dreje sig om emner som regering, parlament, politiske partier og internationale organisationer.

Økonomi er en videnskab, der undersøger, hvordan samfundet producerer, distribuerer og forbruger varer og tjenester. Det omfatter også emner som priser, valuta, arbejdsmarkedet og nationaløkonomi.

International politik handler om de politiske forhold mellem stater og internationale organisationer på verdensplan. Det drejer sig om emner som fred og konflikt, globale udfordringer og samarbejde mellem lande.

Vi håber, at du vil finde vores hjemmeside nyttig og interessant, og at du vil lære mere om samfundsfag gennem vores undersider.

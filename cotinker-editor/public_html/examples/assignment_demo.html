Controlling the Editor

When an editor is open it is possible to perform a number of actions from the remote control guide HTML.

Although all the students in a group will be on the same step in the assignment at the same time, the focus within the editor tabs are individually controllable.

Students can see each others' keyboard cursors and selections.

To make it easier to navigate the code it is possible to provide jump points from this HTML guide to specific points in the code:

  • Highlight a specific range of code based on starting and ending lines and columns
  • Clear the highlight again
  • Potential TODO: Jump the cursor focus to a specific line and column in an already open fragment (may switch tabs to get there)
  • Potential TODO: Jump the cursor focus to the next occourance of the given search text (useful for finding a particular part of the code even after people have edited some code)

The jumps are only carried out on the view tied to the device that opened this guide - you only jump your own cursor, not everyone's.

Assignment step HTML can link to external surveys on the SurveyXact platform. The only requirement is that the survey is set up to link back this domain when the survey is completed - this is a setting in the SurveyXact survey and only the domain is required, not the page.
